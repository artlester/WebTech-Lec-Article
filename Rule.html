The at-rule

A CSS command that starts with the “@” sign. It follows an identifier
    Following at-rules:
  • @charset -specifies the character encoding used in the style sheet. It must be the first element in the style sheet and not be preceded by any character
  • @import - used to import style rules from other style sheets
  • @namespace -is an at-rule that defines XML namespaces to be used in a CSS style sheet
  • @media -used to apply styles based on the result of one or more media queries, which test a device's type, specific characteristics, and environment
  • @supports -lets you specify declarations that depend on a browser's support for one or more specific CSS features
  • @document - restricts the style rules contained within it based on the URL of the document
  • @page - used to modify some CSS properties when printing a document
  • @font-face -allows authors to specify fonts (online url(), and locally local()), to display text on their web pages
  • @keyframes -specifies the animation code. The animation is created by gradually changing from one set of CSS styles to another
  • @viewport -the portion of a device's screen that is available for application use
  • @counter-style -lets you define counter styles that are not part of the predefined set of styles

CSS Rule of Precedence

An HTML element can be the subject of the selectors of multiple style rules, and when multiple CSS rules target the same element applying the same CSS properties there will be conflicts. This problem can be solved by understanding the CSS rule of precedence.

Origin and Importance

The origin and importance is one of the steps of CSS cascade. This both determines which style declaration has the highest priority. Origin refers to where it comes from. Style sheets may have three potential origins namely, author, user, and user-agent.

  • Author - This refers to the person creating the website.
  • User - This refers to the person that is using the website.
  • User-Agent - This is typically the browser.

Importance can be determine by using the !important rule. The !important rule has the highest precedence of all precedence factors, it overrides the previous assigned CSS declarations but the override rule can be override by itself. Example: <style> a { Color: yellow !important; } </style> The sample code above will render all the links on the website with yellow color regardless of the other CSS declarations that can affect the link element.
